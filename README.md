# Rhodes 2026 — Family Vacation Dashboard (Demo)

A fully self-contained, single-file vacation dashboard: scroll-driven route map,
interactive island map with points of interest, flip-card boarding passes,
time-of-day theming with moon phases, an underwater scroll experience,
a Greek word of the day — no build step, no framework, no backend.

**All names, home locations, booking codes, flight numbers and amounts are fictional.**
Photos via Wikimedia Commons (credits in the page footer).

👉 Just open `index.html` — or visit the GitHub Pages deployment of this repo.
